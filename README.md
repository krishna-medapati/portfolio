# Personal Portfolio Website
A full-stack personal portfolio website built to showcase projects and skills.
## Tech Stack
- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Backend: Node.js, Express.js
- Database: MongoDB with Mongoose ODM
## Project Structure
portfolio/
├── backend/
│   ├── models/
│   │   └── Project.js
│   ├── routes/
│   │   └── projects.js
│   ├── server.js
│   └── .env
├── frontend/
│   ├── index.html
│   ├── style.css
│   └── script.js
└── package.json
## Features

- Responsive portfolio with About, Skills, Projects and Contact sections
- REST API backend with CRUD operations
- Projects dynamically loaded from MongoDB
- Dark themed UI

## Setup and Installation

### Prerequisites
- Node.js installed
- MongoDB installed locally
